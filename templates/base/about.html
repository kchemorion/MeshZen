API Documentation

Explore MeshZen's API and integrate it with your applications.

Authentication

MeshZen API requires authentication with Zenodo credentials.

Endpoints

The MeshZen API provides access to the following endpoints:

  • /api/meshes/ - Retrieve a list of meshes
  • /api/meshes/{id}/ - Retrieve a specific mesh by ID
  • /api/meshes/{id}/comments/ - Retrieve comments for a specific mesh
  • /api/meshes/{id}/comments/create/ - Add a new comment to a specific mesh
